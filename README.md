### F-Score 투자전략
재무제표를 기반으로 기업의 수익성, 재무성과, 영업 효율성에 따라 1 또는 0 값을 부여하여 향후 투자 수익률이 높을 가능성이 있는 기업을 선정하는데 사용하는 지표이다.
